Titrated Give-N tests children using a stair-case procedure. E.g., if a child correctly gives 3, the next number tested is 4; if not, the next number is 2. Usually faster, but may test individual numbers fewer times.

Non-titrated Give-N tests a set of numbers (which can be specified below) three times each. Generally slower, but tests individual numbers multiple times.
